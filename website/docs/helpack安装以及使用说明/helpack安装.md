# Helpack安装与使用说明
## Helpack
**Hel Pack**，海拉HelPack，动态化模块发布、托管服务

![](https://tnfe.gtimg.com/hel-img/WX20251027-120047.png)


### 下载hel-micro
[**hel-micro**, 模块联邦 sdk 化，免构建、热更新、工具链无关的微模块方案 ❤️](https://helmicro.com), 在任意目录执行以下命令
```bash
npm i hel-micro
```

### 下载helpack项目
在已有hel-micro项目路径下执行以下命令, 克隆项目并进入helpack目录
```bash
git clone https://github.com/Tencent/hel.git
cd helpack
```
